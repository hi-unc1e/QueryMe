# 介绍
QueryMe，是一款资产信息查询工具，也类似于资产情报查询平台，只不过情报要做到最全！。

渗透中，常常需要查询单个资产（IP、域名）的信息，包括isp运营商，IP归属地、开放端口服务、历史SSL证书、历史解析过的域名等信息，但渗透的时候一个个商平台查，真的好傻啊。
